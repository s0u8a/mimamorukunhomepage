About

みまもるくん（アプリの説明）

GitHub のコミット数やブランチ作成回数を取得して記録するデスクトップアプリです。

概要（何をするアプリか）

みまもるくんは、あなたの GitHub アカウントで認証した後、選択したリポジトリの コミット数やブランチ情報を取得して、ローカルに JSON ファイルとして保存します。 チームの貢献度や活動記録の可視化に便利です。

なぜこのアプリを作ったのか

学生同士のチーム制作では、特定のメンバーにタスクが偏ったり、進め方に対する認識のズレが 少しずつ積み重なって、気づいたときにはチームがうまく回らなくなっている——そんな場面が 少なくありません。みまもるくんは、そうした「チームの崩壊」が実際に起きてしまう前に、 現状に気づき、対策を考えるきっかけを作ることを目指して開発しました。

対象ユーザー

GitHub や Discord を使って制作を進めている学生チームを主な対象としています。

従来のタスク管理アプリとの違い

  • 独自の評価基準である「崩壊度」を算出し、チームが現在どれくらい破綻しかけているかを 視覚的に表示します。
  • GitHub のコミット数や Discord の発言数といったデータを集計し、数値として見える化する ことで、メンバー間の貢献度の偏りにも気づきやすくなります。

主な機能

  • GitHub での OAuth 認証（Device Flow を利用した簡単ログイン）
  • リポジトリを登録・管理（最大5件）
  • 選択したリポジトリからコミット数・ブランチ数などを取得し JSON として保存
  • 取得データは `github-data.json` というファイル名で保存されます

インストール（エンドユーザー向け）

お客さま向けの配布ファイル（ZIP）をダウンロードしてご利用ください。基本操作は以下のとおりです。

  1. ダウンロードした ZIP を任意のフォルダに展開します。
  2. 展開先のフォルダ内にある実行ファイル（例: mimamorukun.exe）をダブルクリックして起動します。
  3. 初回起動時に Windows のセキュリティ（UAC）が表示される場合は許可してください。

起動とログイン

アプリ内の「GitHubでログイン」ボタンを押すと、画面にコードが表示されます。 指示に従ってブラウザでそのコードを入力・承認すると、アプリは安全にGitHubへ接続します（Device Flow を利用した OAuth 認証）。

ログイン後は取得したいリポジトリを選択してデータの取得・保存ができます。

出力されるデータ形式（例）

{
  "owner/repo-name": {
    "commits": {
      "total": 120,
      "byUser": { "user1": 80, "user2": 40 }
    },
    "branches": { "total": 10, "byUser": { "user1": 6, "user2": 4 } },
    "fetched_at": "2026-05-29T07:00:00.000Z"
  }
}

注意事項

  • 本ページはエンドユーザー向けの利用方法を説明しています。開発者向けのセットアップ手順は表示していません。
  • ログイン時に認証が必要になります。表示された指示に従って安全に承認してください。
  • 取得したデータ（例: github-data.json）にはリポジトリ情報が含まれます。取り扱いに注意してください。
