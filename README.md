# みまもる君のダウンロードホームページ
 
ZIPファイルを配布するためのダウンロードページです。GitHub Pagesを利用して公開しています。
 
## サイトURL
 
https://s0u8a.github.io/mimamorukunhomepage/
 
## ファイル構成
 
```
リポジトリ/
├── index.html        # ダウンロードページ本体
├── CSS/
│   └── index.css     # スタイルシート
└── files/
    └── sample.zip     # 配布用ZIPファイル
```
 
## 使い方
 
1. 上記のサイトURLにアクセスする
2. 「ZIPファイルをダウンロード」ボタン(リンク)をクリックする
3. ZIPファイルがダウンロードされる
 
## 注意事項
 
- GitHub Pagesでは1ファイルあたり100MBまでの制限があります。それを超える場合はGitHub Releasesでの配布を検討してください。
- ファイルサイズが大きいと表示・反映まで数分かかる場合があります。
